GUIClick Tool

Purpose: to allow gui automated test authors to use the GUIClick tool to interact with applications using the mouse.

I. Purpose of The Tool

This tool is used to click on anything in the GUI. It simulates mouse clicks of multiple types, including left, right, press (and hold), and release. It can detect an element and click that element, or click a particular pixel location in the indicated window.

II. Data Inputs

This tool expects a data input that follows the variable-value pattern outlined here.

III. Interface

Common Options

Click Location Type: The way in which the click location should be determined. If set to "Point", the click will occur at the specific pixel location. If set to "Element", the click will occur on the element specified by the selectors.

Click Point X Variable (shown only when Click Location Type = "Point"): The variable in the data stream which indicates the window x-axis pixel location where the click will occur. This should be a coordinate that has been output by an upstream tool.

Click Point Y Variable (shown only when Click Location Type = "Point"):  The variable in the data stream which indicates the window y-axis pixel location where the click will occur. This should be a coordinate that has been output by an upstream tool.

X Offset (shown only when Click Location Type = "Element"): X-axis offset for the click operation. This is an offset relative to the element's left-hand border. If you find that a offsets of 0 causes the element to not actually be clicked, a small offset (5 to 10) can fix this.

Y Offset (shown only when Click Location Type = "Element"): Y-axis offset for the click operation. This is an offset relative to the element's left-hand border. If you find that offsets of 0 causes the element to not actually be clicked, a small offset (5 to 10) can fix this.

Click Type: The type of click to perform. Click is a normal, single click. Press will press the specified mouse button down and hold it down until the release click type is performed. Press and release can be used to click and drag.

Button to Click: The mouse button that should be clicked.

IV. Data Outputs

This tool will output the same variable names and values as it received from the input stream with no modifications. This tool will not output any new variable names or values.
