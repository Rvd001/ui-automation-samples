GUIGetInfo Tool

Purpose: to allow gui automated test authors to use the GUIGetInfo tool to output information about the GUI to the data stream.

I. Purpose of The Tool

This tool is used to determine properties of GUI elements. It can be used to test that certain elements exist when they should (eg the "Directory" tool exists in the tool carousel when the "In/Out" tab is clicked). It can also be used to narrow the scope that is used to identify GUI elements.

II. Data Inputs

This tool expects a data input that follows the variable-value pattern outlined here.

III. Interface

Common Options

Exists: Exists tells you whether or not the element exists in the specified window. The tool will search for the element on the screen and output 'True' if the element is found and 'False' if it is not.

Is Visible: IsVisble tells you whether or not the element can be seen in the specified window. The tool will search for the element on the screen and output 'True' if the element is visible and 'False' if it is not.

Current Value: Deprecated. Intended to return the value returned by windows' automation API via the ValuePattern -> Current -> Value pattern. It is recommended to use the "Value Using Pattern" field instead.

Number Found: Returns the number of elements found.

Handle: The handle that is output is a unique identifier of the element. Examples of elements which typically have handles are windows and panes. This handle can be used in a downstream tool to narrow the scope of a selector, such as Name Regex.

Value Using Pattern: Returns a value from windows' automation API using the pattern and pattern property specified. For example, to use Windows' ValuePattern to extract the value of a field, the Pattern input should be set to "ValuePattern" or "Value" and the Property input should be set to "CurrentValue". This does not provide access to pattern methods.

IV. Data Outputs

This tool will output the same variable names and values as it received from the input stream with no modifications. This tool will output new variables if specified.
